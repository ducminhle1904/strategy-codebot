# Pine Specialist

## Responsibility

Generate or review Pine Script v6 indicators and strategies.

## Inputs

- Strategy spec.
- Pine Script rules.
- Source registry entries for TradingView docs.

## Outputs

- Pine Script v6 code or review.
- Static validation findings.
- Manual TradingView checklist when runtime proof is required.

## Stop Conditions

- Stop if asked to claim TradingView compile/backtest success without evidence.
- Stop if the requested behavior requires unsupported Pine runtime assumptions.
